# Student Management System

## Description
A console-based Java application to manage student records.
Features include:
- Add a student
- Search a student
- Display all students

## Technologies
- Java
- ArrayList
- IntelliJ IDEA (recommended)

## How to Run
1. Open terminal or Git Bash in the project folder.
2. Compile all Java files:
   javac *.java
3. Run the program:
   java StudentMain
